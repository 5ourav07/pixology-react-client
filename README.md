Pixology is a personal photographer website. You can see my different services and add a review of these services after login. All the services data are from mongobd.

## Features: 
* Firebase
* React-Router
* React-Bootstrap
* Express
* React-Icon
* React-Hot-Toast
* Vercel
* mongobd
